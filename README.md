# 🐼 Vaccine Panda – Daily Audit PWA

## Install on Phone (Team Members)

Once hosted on GitHub Pages, share the link. Then:

### Android (Chrome)
1. Open the link in Chrome
2. Tap the **⋮ menu** → "Add to Home Screen"
3. Tap **Install**
4. Done — VP Audit icon appears on your home screen!

### iPhone (Safari)
1. Open the link in **Safari** (must be Safari)
2. Tap the **Share button** (box with arrow)
3. Scroll down → tap **"Add to Home Screen"**
4. Tap **Add**
5. Done!

## Files
- `index.html` — The full audit app
- `manifest.json` — PWA configuration
- `sw.js` — Service worker (enables offline use)
- `icon-192.png` / `icon-512.png` — App icons

- ✅ Works offline after first load
